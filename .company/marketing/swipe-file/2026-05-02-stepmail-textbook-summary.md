## 0. すべての出発点：誠司の整理が核心そのもの

> サービスとはレメディ → アウトカム への変化を促すこと。
> その間にプロブレム（問題定義）の壁があり、自力ではレメディ→アウトカムになれない。
> このプロブレムの壁を超えるための手段がサービスである。

教材全体は、**この一文を体系化したもの**。これさえ覚えていれば構造は復元できる。

```
   REMEDY（負の感情・状態）
        │
        │  ← ここに PROBLEM（問題）の壁がある
        │     しかもその問題は「一人では解決できない」（CAN NOT）
        │
        ▼
   サービスを使う
        │
        ▼
   OUTCOME（正の感情・状態）
```

セールスとは、見込み客が **この3点を理解する** こと：
1. REMEDY → OUTCOME になりたい
2. 間にPROBLEM がある
3. PROBLEM は一人では解決できない

→ この3つを「明確・明朗・明快」に伝えれば売れる。それがすべて。

## 1. SALES の定義（1章の中核）

| 文字 | 意味 | 内容 |
|---|---|---|
| **S** | SHINYOU（信用） | コツコツ時間をかけて積み上げる |
| **A** | AKARUI（明るい／明確・明朗・明快） | 販売目的を隠さない |
| **L** | LTV（ライフタイムバリュー） | 一人の顧客への売上を最大化 |
| **E** | EN（縁） | セールスとは縁を作ること |
| **S** | SHINRAI（信頼） | 短期で構築可能。「スゲー」と言わせる |

### 1-1. 信頼 ＞ 信用（順番が重要）
- 信用＝**約束を守る回数を増やすだけ**（コツコツ時間が必要）
- 信頼＝**見込み客に「スゲー」と言わせる／思わせること**（短期構築可能）
- 例：「A君（信用あり）はいい人だけど、B君（信頼あり）と結婚するわ」
- だから **信頼を先に作って、後から信用を積む** のが効率的
- セールスは極論、信用がなくても **信頼があれば** モノは売れる（ここを理解してない人が非常に多い）

### 1-2. 「スゲー」をどう作るか
- 見込み客が知らないことを伝える → だが情報は枯渇する
- 枯渇しないのは **「あなたの視点・物の見方」**
- 「世間ではこう言われているが、私はこう捉えている」「みんな知ってる事を、ああそういう捉え方もあるのかと思わせる」
- テレビのご意見番（マツコ／たけし）が重宝されるのはこの構造
- 視点＝コンテンツ（情報）の肝。

## 2. ターゲット設定の鉄則

### 2-1. ターゲットは「今すぐ客」と「もうちょっとで今すぐ客」だけ

- 今すぐ客は約3パーセント存在（『究極のセールスマシーン』より）
- **がんばれば今すぐ客になる人** をターゲットにすると恐ろしいことに：
  - 今すぐ客に不必要な情報を送ることになる
  - 今すぐ客がうんざりして他で買う
  - 例：エプソンのプリンタを型番決めて買いに行ったのに延々商品説明される苦痛
- ターゲットを絞らないと「全員に売ろうとして全員に売れない」悪循環

### 2-2. 顧客 vs 見込み客の定義
- **顧客**＝お金を払ってくれた人
- **見込み客**＝顧客になる見込みがある人
- 見込み客の中に顧客がいる → 時間ベクトル「見込み客 → 顧客」が存在

## 3. NOT × 4（読まない・信じない・行動しない＋NOT UNDERSTAND）

コピーライティング世界で有名な「3 NOT」に上野氏が一つ追加：

| NOT | 内容 |
|---|---|
| NOT READ | 読まない |
| NOT BELIEVE | 信じない |
| NOT ACT | 行動しない |
| **NOT UNDERSTAND** | **理解しない**（最難関） |

### 3-1. なぜ NOT UNDERSTAND が最難関か
**人それぞれ使っている日本語が違う**から。
- 「ちょっと遅れる」→ Aさんは5分／Bさんは15分／Cさんは2分
- 「めちゃくちゃ怒ってる」→ 実際出てみたら大して怒ってない
- 「いつも」「毎回」→ 実は2〜3回しか起きてないのに「いつも」と言う（**盛り癖**）

→ 副詞・形容詞は人によって定義が違う。セールスでは致命的。
→ 逆に言えば、**こちらが定義をしてしまえば強烈な武器になる**。

### 3-2. 絶対言葉
強烈にプッシュする時の修飾語。人によって最上級の言葉が違う：
- 「絶対」「かならず」「何が何でも」「強く」「これだけは」「死んでも」
- 「絶対が最上級」の人に「かならず買って」と言っても全然響かない
- → **事前に「絶対言葉」を比喩で定義**しておく
- 例：「私が『何がなんでも』と言う時は、崖っぷちで子供の手を離さないレベルの『何がなんでも』です」

→ クロージング時の切り札。連呼すれば価値が落ちる。

## 4. 思い切ってシンプルに

- 過去1000名以上の添削で、**高学歴の人が書くステップメールほど売れない**
- 理由：複雑なロジック「こうしてああして、こうだからああなって、だからこういう場合は…」
- 読者は3通目を読む頃には数日経過＝**昼飯レベルの記憶**しか残らない前提で書く

### 4-1. 失敗例 → 成功例の修正
- ❌ 「あなたが稼ぐためには情報起業する必要があります。情報起業とは商品を作ってメルマガを書いてセールスすることです」
- ✅ 「あなたが稼げないのは方法を知らないからです。方法を知ればこのように稼げます」
→ これで億の売上に。**シンプルなロジックほど効く**。

## 5. PROBLEM（問題定義）の作り方 — セールスの肝

### 5-1. 切り込み方
> ❌「あなたのステップメールで売れないのは、人を思いやる気持ちが足りなくて独りよがりだからです」
> （誰も納得しない／反発される／信頼が築けない）

> ✅「あなたのステップメールで売れないのは、このステ教の1通セールスパターンを知らないからです」
> （ズバッと切り込み、はっきり明確に理解させる）

→ 「あ！なるほどね！だからか！知らなかったー！」を作る。

### 5-2. 最重要：「一人では解決できない」(CAN NOT) を作る
PROBLEM が定義できても、見込み客が「自分でやろう／できるかも」と思ったら売れない。

例：「人間関係でうまくいかないのは、相談する人がいないから」と定義
→ 読者「そうか、信頼できる部長に相談しよう！」と独力解決される

→ だから **「誰に相談しないといけないか？」** まで定義する必要がある。
→ ★PROBLEM を定義する時は **CAN NOT（自分一人ではできない）まで明確化** することが肝中の肝。

## 7. 情報発信テーマの決め方：8つの宝島

人生は8つで成り立つ：

```
【人間関係】          【お金】
  自分                 仕事
   ├ 趣味             資産
   ├ 旅行
   ├ 健康
  家族
  交友関係
```

→ 突き詰めれば **「人間関係」と「お金」の2つ** だけ。
→ 8つから1つ選んで明確化しないと情報配信がブレる。

例：上野氏の前職（講習販売）は「健康といい気分」をテーマに → だから宝石・呉服も売れた（健康だけだと売れない）。

## 8. 【最重要】セールスの目的（1章のクライマックス）

### 8-1. 目的は2つ。明確に分ける

| 主体 | 目的 |
|---|---|
| **販売者（あなた）** | 販売目的を正々堂々と明かすこと。明確・明朗・明快に |
| **購入者（見込み客）** | あなたの情報を受けることでメリットがある／購入でメリットがあると分かる |

→ この2つの目的が **一致する** ようメッセージを送る。

### 8-2. 販売目的を隠すと何が起きるか
- 情報配信は無料という思い込みが強い
- セールス嫌われを恐れて販売目的を不明確にする人が多い
- 「販売しません」と言いながら販売している人ばかり
- → これを続けると **「いつも情報ありがとう」「楽しみにしてます」的な“買わない読者”** に育つ
- 残念ながらそれは**自分が買わない読者に教育してきた**結果
- 読者の質が悪いのではない。**自分が育て上げた**

### 8-3. 正しい順序
- 販売目的を伝える → 購入意思がない人は読まなくなり去っていく
- 残った人＝**いいものがあれば購入するかもしれない読者ばかり**
- 販売目的を繰り返し伝える＝知らない間に教育されている

→ **「目的に合わない場合は解除を促してください」** これが大事。

## 9. 第1章まとめ（暗唱レベルで覚える）

> SALES の定義の通りに、信用を得て、明るく（明確・明朗・明快）にメッセージを送り、LTV を最大化し、読者と良縁を繋ぐことで信頼を得る。
> 販売目的を明確にして、読者の目的と販売とを合致させる。正々堂々とセールスする。
> セールスの際には、絶対言葉を事前に読者に定義しておくことで、強く、絶対言葉で勧める。
> 究極、セールスとは販売者が「このサービス絶対おすすめ」というメッセージを繰り返し伝えることで、見込み客が購入するだけのこと。

## 10. 第2章 ステップメールの構造

### 10-1. 核心：興味ある人にしかセールスしない
- ステップメールの最大の武器は **クリック分岐URL**
- 「○○サービスというものをご紹介します。興味ありましたら、下記リンクをクリックして下さい」 → クリック分岐URL → これだけ
- クリックした人 = 今すぐ客／もうすぐ今すぐ客 → ガンガンセールス
- クリックしない人 = 興味なし → セールスしない（買う確率は限りなく0）

### 10-2. 売ってから理解しよう
- 理解してから売るのではない
- まずやってみて、実践で売れたら理解できる
- 順番が逆だと永遠にステップメールが書けない

### 10-3. 追撃
- 単純にセールスを繰り返す＝追撃
- セールスレターへの誘導を繰り返す
- 何通書いてもいい。ターゲット（今すぐ客／もうすぐ今すぐ客）が迷っているだけ
- 売れなかった日があっても勝手に「もう売れない」と未来を予知する預言者になるな（実例：1回目0／2回目2／3回目2／4回目1／5回目0／6回目5＝計10個）
- 購入者は配信停止、解除リンクを使う

### 10-4. 関心 ＜ 興味（最重要方程式）

| 関心 | 興味 |
|---|---|
| 心が**点**で少しだけ何となく引っかかった | **五感で味わって興じている** |
| まだただの点 | 五感で触れる立体 |

**関心を増やす → 線 → 面 → 形 → 立体 → 触りたい（興味）**

### 10-5. 関心を増やす方法
1. **ダーツの的にどんどん投げる**：情報を多く知っておく → どれかが刺さる
2. **関連情報を送る**：刺さった情報に関連する情報を送る → 点が線になる
3. **複数の線を交わらせる**：A情報とB情報がつながる → 面になる
4. **読者との関係を伝える**：その情報が読者と関係があるとメッセージで伝える
5. **繰り返す**：面が立体になる → 興味になる

### 10-6. 解除リンクを正々堂々と置く
- 「いつでもご自由に解除できますので、ご安心ください」
- 「一度解除したらブラックリストになる」「解除したらお得な情報が届かない」と書くのは気持ち悪い
- 解除を促すと、残った人＝興味がある人にだけ自信を持ってセールスできる

### 10-7. 注意：時限式リンクの使い方
- 「3日限定」のリンクは効く（限定性／後回しNG教育）
- ただし「延長します」をやると一気に信用が崩れる（狼少年）
- 短期売上 vs 長期良縁は **長期良縁を取る**

## 11. 第3章 記事の書き方

### 11-1. 記事の構成
件名 → あいさつ文 → 導入文 → 本文 → まとめ → 編集後記（追伸）

### 11-2. 件名 = 命
- 「自分なら間違いなく開く」基準で書く
- 普段思わず開いた件名・電車広告のコピー・本のタイトルを **書き留める習慣**
- 関心＜興味 → 見込み客の関心は何か？ 興味は何か？ ここを徹底的に考える

#### ★感情あおり系（釣り件名）は使わない
- 例：「もう死にたいです」「殺してやりたいです、あいつを」「あなたをゆるしません」のような **感情あおり系**
- 開封率は確かに上がるが、本文と関係ないので **読者を不快にする＝敵を作るだけ**
- 本文とテーマの整合性が取れていれば救いがあるが、無意味なので **使わない**
- 「コピーライティングは突飛なことを書かないと」「キャッチーな言葉を書かないと」は誤解。**ターゲットさえ正しく理解できていれば、件名はそんなに難しくない**

### 11-3. 名前挿入機能
- 「YOU」が反応取れるキーワード（読者自身が「自分の事だ」と認識した時）
- フルネーム不要（苗字だけ）
- 普段は「あなたあなた」、ここぞで「○○さん」とドキッとさせる
- 使いすぎ禁物

### 11-4. 理想のターゲット = 自分自身
- ペルソナ不要
- 読者は自分に似た人が一番商品を購入する
- 自分の趣味・嗜好・年齢・出身地・住所・学歴・価値観を **早々と開示** して、似た人を集める

#### ★1通目から自己紹介を長々やるのはNG
- メルマガ1通目で「お涙ちょうだい」の自己紹介を長々やる人がいるが、**読者は引く**
- リアル店舗に初めて入った時にいきなり店員からそれをやられたら引くのと同じ
- まず **「このメルマガは読んだらメリットがあるな」と思ってもらう** のが先
- パーソナル情報（趣味嗜好・出身地・価値観）は **本文ではなく編集後記で** 出す
- アイデア：詳細な自己紹介ページをブログに作っておき、編集後記からリンクを貼る → クリック率で「自分という人間にどのくらい興味を持ってくれたか」を計測できる

### 11-5. あいさつ文 = 趣味嗜好の刷り込み
- 「こんばんは、上野です。今日もソーメンチャンプル食べてしまいました」
- 「ソーメンチャンプル＝上野」と刷り込まれる
- 読者がソーメンチャンプル食べた時に上野を思い出してしまう（思い出したくなくても）

### 11-6. 編集後記 = FAN化のキー（LTV直結）

> 編集後記まで読む人 = あなたの FAN

#### 4要素
1. どんな感情を持っているのか
2. どんな欲求を持っているのか
3. 読者にどうなって欲しいか／なって欲しくないか
4. 自分はどうなりたいか／なりたくないか

→ 1, 2 を **意図的に** 喜怒哀楽として見せる。
→ ペット・子供のネタも◎。

### 11-7. 戦略：欲求はいつも叶っている！
> あなたの欲求はいつも叶えられている

これを刷り込む。
- 「温泉にゆっくり行きたいなあ」を数回 → 数回後「ついに行ってきましたあww」と幸せ感全開
- これを **必ず達成できる欲求** で繰り返す＝出来レース、ただし本物
- 結果：「いいなあ／うらやましいなあ／**スゲええなあ**」 → SHINRAI（信頼）になる

### 11-8. 導入文の3レベル
| レベル | 内容 | 用途 |
|---|---|---|
| **LEVEL 1** | 「今日このメールでお伝えしたいことは○○です」 | ストレート、ライティング苦手な人の必殺技。すべて「（〜したい）」形 |
| **LEVEL 2** | LEVEL 1 + その理由 | 万能。「みなさんが興味あるから」「先日約束したので」を理由にすれば必ず使える |
| **LEVEL 3** | 興味付け（必殺技） | セールスが楽しくなる |

### 11-9. LEVEL 3 興味付けの基本構造
```
もし、あなたの悩みが解決できる、しかも簡単にできる方法があれば
それを知りたいですか？

この方法を実践したAさんは
こうなりました
ああなりました
さらにこうなったんです

あなたも、もし上記のようになれたらいいと思いませんか？

そういう方法を今日のメールではお伝えしたいと思います。
```

→ ① 疑問文で問いかける ② 証拠や事例を示す ③「あなたもそうなりたい？」と確認

### 11-10. まとめ
- 「最後にこのメールでお伝えしたかった事をまとめます」
- 「最後になりますが、このメールで一番伝えたかったことは…」

### 11-11. 補足：小川氏の実践参考
- **エニアグラムセールス**：9タイプの価値観キーワードで反応が劇的に変わる（クリック率2倍、成約数5倍）
  - キーワード例：一生懸命／癒やし／目標達成／自分らしさ／マイペース／ワクワク／知識／原理原則／信頼／現実的／理想／夢
- **自分の体験からメッセージを伝える**：
  - (1) 湯浅醤油 → 「ちょっとした違いを大事に」
  - (2) コーヒー自家焙煎 → 「やってみると意外と簡単」
  - (3) サルサのダンス → 「手を動かさないと身につかない」
- **すげーのカンタンな作り方**：継続している姿だけで魅力的。「メルマガを続けている」だけでスゲー

## 17. 誠司への一行サマリー

> **REMEDY → PROBLEM（一人では超えられない壁）→ OUTCOME。サービスはその壁を超える手段。これを明確・明朗・明快に、繰り返し、正々堂々と伝える。それだけ。**

これさえ覚えていれば、ステップメールも、LPも、対面営業も、SNSも、すべて同じ構造で組める。

教材全体を通じて何度も繰り返されるメッセージ：
- 「シンプルにできるかどうか」
- 「明確・明朗・明快」
- 「正々堂々と」
- 「繰り返し、繰り返し、繰り返し」
- 「目的を明かす」

これらは互いに同じことを別の角度から言っている。
