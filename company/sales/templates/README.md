# templates/ — アウトリーチ文面テンプレ

## 用途
各商材・用途別のメール文面、ピッチ資料のベーステンプレを管理する。
AI はこれを参照して各社向けにパーソナライズを行う。

## 命名規則
`[用途]-[バージョン].md`

例:
- `impression-training-v1.md` — 印象研修 初回アプローチ
- `impression-training-followup-v1.md` — フォローアップ
- `give-take-initial-v1.md` — Give&Take 協会 初回

## 推奨フォーマット

```markdown
# [用途] テンプレ v1

## 想定ターゲット
- 業種:
- 企業規模:
- 担当者層:

## 提供価値（訴求ポイント）
- 主訴求:
- 副訴求:
- 差別化:

## 件名案
- 案1:
- 案2:

## 本文構成
### 冒頭（パーソナライズ部分）
- 各社の特徴を1〜2文で言及
- なぜ貴社にこの提案をしているか

### 提供価値の説明
- スキームの概要
- 企業側メリット（負担ゼロ、社員育成、etc.）

### CTA（アクション誘導）
- 資料請求 / オンライン面談 / 日程調整リンク

### 署名・法令対応
- 送信者名・会社名・住所
- 広告表示
- 配信停止方法

## 禁止事項
- 誇大表現
- 未確認情報
- 過度な緊急性の演出
```

## バージョン管理
- 大きな変更は v1 → v2 として新ファイル作成
- 反応率が大きく変わったら記録し、勝ちパターンを残す
