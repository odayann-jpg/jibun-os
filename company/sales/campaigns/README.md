# campaigns/ — キャンペーン・施策記録

## 用途
月別・施策別のアウトリーチ活動の記録と振り返り。
パイプラインの状況、反応率、改善点を一元管理する。

## 命名規則
`YYYY-MM-[施策名].md`

例:
- `2026-05-リフレイム-不動産-pilot.md`
- `2026-06-印象研修-BtoB営業-拡大.md`

## 推奨フォーマット

```markdown

## 目的・KPI
- 目的:
- 目標件数:

## ターゲット条件

- 企業規模:
- 地域:
- 除外条件:

## 実施内容
- リスト作成日:
- 送信期間:
- 送信チャネル: メール / 問い合わせフォーム / 郵送
- 使用テンプレ: [templates/xxx-v1.md]

| 商談化件数 |  |  |  |
| 受注件数 |  |  |  |

## 改善点

```

## 振り返りタイミング
- 施策終了時に必ず振り返る
- 月末にその月の施策総括を作成
- 四半期ごとに勝ちパターンを templates/ に反映
