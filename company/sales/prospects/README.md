# prospects/ — 候補企業リスト

## 用途
AI で作成した候補企業リストを業種・キャンペーン別に管理する。

## 命名規則
`YYYY-MM-DD-業種-list.md`

例:
- `2026-04-28-不動産-list.md`
- `2026-05-05-BtoB営業-list.md`

## 推奨フォーマット

```markdown
# 候補企業リスト: [業種] ([作成日])

## 作成条件
- 業種:
- 従業員規模:
- 地域:
- その他条件:

## リスト

| No | 会社名 | URL | 従業員 | 業種詳細 | ニーズ度 | 問い合わせ先 | ステータス | 備考 |
|----|--------|-----|--------|----------|----------|-------------|----------|------|
| 1  |        |     |        |          |          |             | prospect |      |
```

## ステータス
- prospect: 未アプローチ
- contacted: アプローチ済み
- responding: 反応あり
- disqualified: 非対象（送信先として不適切と判断）
