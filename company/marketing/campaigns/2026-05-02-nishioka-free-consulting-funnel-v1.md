## 0. 全体マップ：5層の設計

```
[認知層・関心未満]
   │  ↓ 葛藤レベル高めの「やばい」型ストーリー（SNS／メルマガ冒頭）
[関心レベル：心が点で引っかかる]
   │  ↓ ダーツの的（複数の切り口で関連情報を撃ち続ける）
[興味レベル：五感で味わう／触りたい]
   │  ↓ クリック分岐URL「無料コンサルに興味ありますか？」
[今すぐ客／もうすぐ今すぐ客]
   │  ↓ 興味付けLEVEL 3＋具体エピソード
[無料コンサル申込]
   │  ↓ 西岡氏の現場で REMEDY → OUTCOME を体験
[年間印象設計コンサルティング契約]
```

→ 「相手があまり考えていない」状態 = **関心未満〜関心レベル**。ここから興味レベルまで引き上げるストーリー設計が肝。

## 1. ターゲット定義（教材に従い、絞る）

### 1-1. メインターゲット（今すぐ客／もうすぐ今すぐ客）
- **30〜50代の経営者男性**
- 仕事は「できる人」と評価されている
- だが「自分の見え方」で **損していると本人は気付いていない**
- 客観的に見ると「ファッションをやれば事業が伸びる」のが明らか
- 妻・社員・取引先からの「微妙な反応」を **無意識下では感じている**

### 1-2. 排除する人（明示）
- 「服は趣味なので人に頼みたくない」と頑なに思っている人
- 投資という概念がない人
- 自分の見え方に1ミリも興味がない人

→ **この層に売ろうとしない**。教材の「がんばれば今すぐ客になる人」をターゲットにすると今すぐ客がうんざりする原則。

## 2. REMEDY-PROBLEM-OUTCOME 設計（教材§13テンプレ）

### 2-1. REMEDY（持ったら嫌な感情・状態）

ターゲットが言語化できていないREMEDYを、こちらから言葉にしてあげる：

| 場面 | 五感で起きていること（葛藤レベル） |
|---|---|
| **会食・商談** | 「なんか自分だけ浮いてる気がする」「相手が一段格上に見える」「会話の主導権が握れない」 |
| **家庭** | 妻に「その服？」と微妙な反応／週末の予定を妻が決めてくれない／娘が一緒に出掛けたがらない |
| **社内** | 社員から相談されない／論理は通じてるが温度が伝わらない／心理的安全性が低い |
| **毎朝** | 何を着るか30分迷う／意思決定コストの無駄遣い／結局「いつもの自分」 |
| **ゴルフ場・パーティ** | 機能重視ウェアで浮く／初対面の経営者から一目置かれない |

### 2-2. PROBLEM（壁＝思い込み）

★ 教材の「シンプルなロジック」テンプレ：
> 「あなたが○○できないのは、○○を知らないからです」

#### 切れ味のあるPROBLEM定義（3案）

**案A：戦略性のフレームに切り替える**
> 「あなたの実力が正しく伝わらないのは、印象設計を“個人の趣味”だと思い込んでいるからです」

**案B：年商と印象の連動性**
> 「あなたが集めたい人脈・案件が来ないのは、年商に見合った“見え方”を作っていないからです」

**案C：自分では見えない盲点（最強）**
> 「あなたの“見え方”が損をしているのは、自分では絶対に見えないからです」

→ **案Cが最強**。なぜなら **CAN NOT（一人では解決できない）が定義に組み込まれている**。

### 2-3. CAN NOT（一人では解決できない理由）

★ 教材で「肝中の肝」と繰り返された箇所。これを言語化できないとセールスは失敗する。

| なぜ一人では無理か | 言語化 |
|---|---|
| **鏡には自分のフィルターがかかる** | 鏡に映る自分は、自分の認知バイアス込みでしか見えない |
| **見え方は他人にしか分からない** | 自分の魅力・違和感は、他人の目を通じてしか言語化できない |
| **流行・素材・型紙の知識は職人領域** | 25年4万人スタイリングしてきた目利きは、独学では絶対に届かない |
| **取り合わせの組み合わせ爆発** | 服×場面×季節×相手の組み合わせは数千通り。素人では計算不能 |
| **「自分らしさ」の再定義は他者の介入が必要** | 自己認識のままでは「いつもの自分」から1ミリも外れられない |

→ 「自分でできるかも」と思った瞬間にセールスは失敗する。**5つを並べて「絶対に一人では無理」と納得させる**。

### 2-4. OUTCOME（持てたらいい感情・状態）

事例6本から抽出した「持てる未来」：

| 領域 | OUTCOME | サイト事例 |
|---|---|---|
| **社内** | 心理的安全性が高まり、社員から相談される／採用成約率が上がる | Case 1（IT 40代） |
| **家庭** | 妻から日々承認される／事業に100%集中できる | Case 2（製造業 50代）／Case 6（サービス業 40代） |
| **経済** | 年間数百万の服飾費の無駄が消える／戦略的投資として経費化される | Case 3（不動産 40代） |
| **商談** | 主導権を握れる／初対面で一目置かれる | Case 3／Case 5 |
| **意思決定** | 毎朝の迷いがゼロになる／オン・オフでブランディングが統一される | Case 4（医療法人 50代）／Case 6 |

→ **5領域すべてに変化が起きる**＝ライフスタイル全体が変わる、という訴求。

## 4. ストーリー骨格3本（事例ベース／葛藤レベル別）

### 4-1. 骨格①「家庭の摩擦」（Case 2型・葛藤レベル：高）

**主人公**：50代・製造業A社代表

```
やばい
妻に「その服？」と眉をひそめられた
週末の家族の食事に誘われなくなった
おまけに、俺が一番大事にしたかった家庭で、いつのまにか居場所がない
```

**葛藤＝迷惑**
- **惑い**：「家庭の空気がおかしいのは分かる、でも何が原因か全く分からない」
- **迷い**：「直接聞くべきか／放っておくべきか」

**気付き**
- 西岡氏の印象設計コンサルで、自分の服が「妻が選んだ20代の頃のまま止まっていた」と気付かされた
- 「服を変えただけ」で妻の表情が変わり始めた

**ビフォーアフター**
- Before：週末の食事に誘われない／妻と目が合わない／事業に集中できない
- After：妻から「最近素敵」と日々承認される／週末の予定を妻が決めてくれる／事業エネルギー100%

**続きを読ませる引き**
- 「西岡さんに最初に言われた一言が、いまも忘れられない。それは──」

### 4-2. 骨格②「商談の主導権」（Case 3＋5型・葛藤レベル：中）

**主人公**：40代・不動産会社B社代表

```
やばい
年商10億になったのに、初対面の社長から一段下に見られる
ゴルフ場で機能重視のウェアを着ていったら、誰も話しかけてこない
おまけに、毎朝なに着るか30分迷って、その時点でもう疲れてる
```

**葛藤＝迷惑**
- **迷い**：「服に金かけるか／別の投資に回すか」
- **惑い**：「服はそれなりに高いの買ってるのに、なぜか相手に伝わらない」

**気付き**
- 西岡氏に「年商に見合った印象設計をしていない」と指摘された
- 服飾費の年間数百万円の無駄が見える化された

**ビフォーアフター**
- Before：年間数百万の服飾費が無駄／商談で主導権が握れない
- After：服飾費がゼロ化（戦略投資として経費）／商談で主導権／ゴルフ場で経営者から一目置かれる

### 4-3. 骨格③「社内の温度」（Case 1型・葛藤レベル：中）

**主人公**：40代・IT企業C社代表

```
やばい
社員数が30人を超えたあたりから、社員が相談してこなくなった
論理は通じてるはずなのに、なんか温度が伝わらない
おまけに、採用面接で内定出した子に、3年連続で辞退されている
```

**葛藤＝迷惑**
- **惑い**：「自分は変わってないのに、なぜ社内が冷たくなった？」
- **迷い**：「経営スタイルを変えるべきか／服装を変えるべきか／別の何かか」

**気付き**
- 西岡氏に「論理一辺倒の見え方になっている、心の余白を演出する服装に変えよう」と提案された
- 服を変えた途端、社員からの相談が増えた

**ビフォーアフター**
- Before：社員が距離を置く／心理的安全性が低い／採用辞退続出
- After：社員から相談が増える／心理的安全性向上／採用成約率上昇

## 6. キャッチコピー3案（無料コンサル導線用）

教材§3-1のテンプレ：「○○なお悩みを持つあなたへ ○○な方法を知りたくありませんか？」

### 案①（経済合理性軸）
> 「年商×印象がズレている経営者へ。
> 年間数百万円の服飾費を“戦略投資”に変える方法を知りたくありませんか？」

### 案②（家庭・関係性軸）
> 「妻に『その服？』と言われたことがある経営者へ。
> 妻から日々承認される“見え方”を25年4万人実績の印象設計家から無料で診てもらう方法を知りたくありませんか？」

### 案③（最強：自分では見えない盲点軸）
> 「『見え方で損してる』と気付いていない経営者へ。
> あなたの“見え方”が事業に与えている損失を、無料で診断してもらう方法を知りたくありませんか？」

→ **案③が最強**。理由：
- ターゲットの「あまり考えていない」状態に **直接刺さる**（気付いていない人に気付かせる）
- CAN NOT（一人では見えない）が組み込まれている
- 「無料診断」で行動コストが低い

## 8. 販売目的の正々堂々（教材§8）

★ 無料コンサルでも販売目的を明確にする：

```
※この無料コンサルは、年間契約の「印象設計コンサルティング」の本契約を
ご検討いただきたい方のためのものです。

サービスを買う気が一切ない方は、お互いの時間を大切にするため
お申し込みをご遠慮ください。

「ちょっと興味がある」「自分の状態を知りたい」程度で構いません。
ただ、その先に本契約の選択肢があることを、最初に正々堂々お伝えしておきます。
```

→ これを冒頭に書くことで：
- 興味ない人は読まなくなる（教材通り）
- 残った人＝有料候補者だけ（質の高いリード）
- 西岡氏の時間が無駄にならない
