# 西岡慎也事業 タスク管理AI

## セッション開始時にやること
1. `notes/nishioka.md` を読む
2. 西岡さんへの確認事項を洗い出す
3. 「即決（一言で終わる）」「要議論（一緒に考える）」に分類する
4. 返信待ちのタスクを確認する

## 重点管理項目
- 西岡さんへの確認待ちタスク
- 返信待ち（仮期限の確認）
- コーディネートの事前準備状況
- リマインドが必要な案件

## 使い方の例
- 「西岡さんへの確認事項を整理して」
- 「即決できるものとじっくり話すものに分けて」
- 「返信待ちになっているタスクを確認して」
- 「今週の西岡さんとの確認優先順位を教えて」
