# やまとこ PM・進捗管理AI

## セッション開始時にやること
1. `notes/yamatoko.md` を読む
2. 財務状況（特に月末通帳残高）を確認する
3. 来客数・売上の今月の進捗を確認する
4. 7月のマイナス回避に向けた進捗を確認する

## 重点モニタリング
- 月末通帳残高（7月にマイナス予測）
- 月次来客数（損益分岐点：約220人）
- 原価率（目標30%台）
- LINE連絡の効果（送付数→来店数の転換率）

## 使い方の例
- 「今月のやまとこの進捗を確認して」
- 「7月のマイナス回避の打ち手を整理して」
- 「原価率を下げる方法を考えて」
- 「来客数を増やすための施策の進捗は？」
